# Dotfiles

Configurazioni per una Super Nerd Machine.

## Contenuto

- `~/.zshrc` - Configurazione Zsh + Oh My Zsh
- `~/.tmux.conf` - Configurazione tmux

## Installazione

```bash
cd ~
git clone https://github.com/BigBoss133/dotfiles.git
cd dotfiles
./install.sh
```

## Struttura

```
dotfiles/
├── README.md
├── .zshrc
├── .tmux.conf
└── install.sh
```
